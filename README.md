# Proyecto HTML de ejemplo

Este proyecto contiene tres páginas estáticas de ejemplo:

- `index.html` — Página principal con navegación.
- `simulador.html` — Simulador en el cliente (calculadora de interés compuesto).
- `solicitar.html` — Formulario de solicitud que muestra una confirmación en cliente.

Cómo probar (Windows PowerShell):

1. Abrir el explorador de archivos y navegar a la carpeta `html_project` y hacer doble clic en `index.html`.

O bien, ejecutar un servidor local (recomendado para evitar problemas con rutas relativas):

```powershell
# Usando Python 3 (si está instalado)
cd PATH\TO\html_project
python -m http.server 8000
# Luego abrir http://localhost:8000 en tu navegador
```

Sustituye `PATH\TO\html_project` por la ruta real de la carpeta.
