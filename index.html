Bienvenido

Este proyecto contiene tres páginas estáticas: `index.html`, `simulador.html` y `solicitar.html`. Usa la navegación arriba para moverte entre ellas.

  • `index.html` — página principal (esta).
  • `simulador.html` — ejemplo de una pequeña aplicación en cliente (simulador).
  • `solicitar.html` — formulario de envío (sin backend, confirmación en cliente).
Abre estos archivos en tu navegador o ejecuta un servidor local para probar.
